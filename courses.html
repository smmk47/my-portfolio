Courses

Semester 1

  • CS1002: Programming Fundamentals - Introduction to the basics of programming, including variables, control structures, data types, and problem-solving techniques.
  • CS1004: Object Oriented Programming - Focuses on the principles of object-oriented programming, including classes, objects, inheritance, and polymorphism.

Semester 2

  • CS1005: Discrete Structures - Covers mathematical structures that are fundamentally discrete rather than continuous, including sets, relations, graphs, and combinatorics.
  • CS2001: Data Structures - Introduction to data structures such as arrays, linked lists, stacks, queues, trees, and graphs, emphasizing their implementation and application.

Semester 3

  • CS2005: Database Systems - Explores database design, modeling, normalization, and SQL for managing data effectively in various database systems.
  • CS2006: Operating Systems - Covers the principles of operating systems, including process management, memory management, file systems, and system calls.

Semester 4

  • CS2009: Design and Analysis of Algorithms - Focuses on algorithm design techniques, analysis of algorithm efficiency, and complexity theory.
  • CS3001: Computer Networks - Introduction to computer networking concepts, including network architecture, protocols, and data communication.

Semester 5

  • CS3004: Software Design and Analysis - Covers software design methodologies, software development life cycle, and techniques for analyzing software requirements.
  • CS3005: Theory of Automata - Explores formal languages, automata theory, and computability, including finite automata and Turing machines.
  • CS3006: Parallel and Distributed Computing - Covers concepts and techniques for designing and implementing parallel and distributed systems.
  • CS3009: Software Engineering - Focuses on software development processes, methodologies, project management, and quality assurance.
  • CS4039: Software for Mobile Devices - Covers the design and development of software applications for mobile devices, including user interface design and mobile operating systems.
  • CS3002: Information Security - Explores principles of information security, including cryptography, network security, and risk management.
  • CS4001: Professional Practices in IT - Focuses on professional ethics, standards, and practices in the IT industry, including project management and teamwork.
  • CS4032: Web Programming - Covers web development technologies, including HTML, CSS, JavaScript, and server-side programming.
