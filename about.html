Objective: To design and develop a web-based application that addresses creative writing challenges, enhancing productivity and storytelling for English and Urdu writers.

Problem Statement: Writers face creative blocks, disorganized workflows, and limited grammar proficiency, hindering overall productivity.

Outcomes:

  • AI-assisted writing tools for overcoming creative blocks and grammar limitations.
  • Modifiable structured writing environment for efficient workflow management.
  • Collaborative platform for seamless multi-author projects.
  • Comprehensive suite of tools for character development, plot management, and publishing.

Key Features:

  • Workspace Dashboard with chapter-wise writing, outline management, and notes creation.
  • AI-powered grammar checker and fine-tuned chatbot assistant.
  • Collaborative writing with real-time syncing and project role management.
  • Language options for English and Urdu.
  • Publishing options with customizable privacy settings.

Technologies Used:

  • MongoDB for flexible data storage.
  • Express for backend routing and API requests.
  • React for dynamic frontend development.
  • Node.js for efficient server-side logic.
  • Hugging Face for fine-tuning and deploying the language model.
